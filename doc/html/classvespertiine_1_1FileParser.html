Detailed Description

Parses COM S 327 DIF files to a vector of maps of string pairs.

The constructor expects a string of a filepath. The file can be parsed under the following assumptions:

  1. Possess a header on the first line of the format "RLG327 [entity_type] [file_type] [version_number]"
  2. Possess entities wrapped in BEGIN [ENTITY_NAME] ... END tags on their own lines.
  3. Possess keys of at least two capital letters followed by a newline or a value. 3a. If followed by a newline, the multiline values must end with a [.] on its own line.
